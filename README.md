# Final-Project-Python
   Sales in Retail Store

The aim of this project is to clean, process, and visualize a real-life dataset using Python.
The project focuses on improving data quality, handling missing or incorrect values, and generating meaningful visualizations to understand patterns and insights.
       >>  Importing of Libraries and Data loading.
       >>  Understanding the Dataset.
       >>  Identification and Handling of Missing Values.
       >>  Sorting of customer ID.
       >>  Detecting and Removing Duplicates.
       >>  Data Visualization.
       >>  Add new columns for Discount Percentage, Price After Discount.
       >>  Data Visualization (Matplotlib/Seaborn).
       >>  Data insights.
